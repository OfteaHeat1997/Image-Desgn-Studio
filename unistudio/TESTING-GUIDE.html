0. Pre-requisitos Antes de Testear

Antes de probar cualquier modulo, necesitas verificar que tu entorno esta configurado correctamente.

A. Variables de Entorno (.env.local)

Abre tu archivo .env.local en la raiz del proyecto y verifica que estas variables existen:

VariablePara Que SirveModulos que la UsanComo Obtenerla
REPLICATE_API_TOKENAPI de IA para procesar imagenes/videoBG Remove, BG Generate, Shadows IA, Inpaint, Outpaint, TryOn, Model Create, Ghost, Jewelry, Video, Avatarreplicate.com/account
FAL_KEYProveedor alternativo de video IAVideo (LTX, Wan 2.5, Kling, Minimax)fal.ai/dashboard
ANTHROPIC_API_KEYClaude IA para texto y planificacionBG Generate (optimizar prompt), Director Creativo, AI Agent, Ad Creator, Video Autoconsole.anthropic.com
FASHN_API_KEYTry-on especializado para ropaTryOn (proveedor FASHN)fashn.ai
DATABASE_URLBase de datos PostgreSQLHistorial, Galeria desde DBTu proveedor PostgreSQL
Si NO tienes una variable, los modulos que la usan mostraran un error. Eso es NORMAL — no significa que el codigo esta roto. Significa que necesitas la API key.

B. Checklist de Preparacion

  • ✓El servidor esta corriendo (npm run dev) y puedo abrir http://localhost:3000
  • ✓Tengo al menos REPLICATE_API_TOKEN en mi .env.local
  • ✓Tengo una imagen de prueba lista (JPG o PNG de un producto, ~1MB)
  • ✓Abri /api/health en el navegador y vi el JSON de estado
  • ✓Tengo la consola del navegador abierta (F12 → Console) para ver errores

C. Imagen de Prueba Recomendada

Para mejores resultados, usa una foto de producto con fondo simple. Tamaño ideal: 1000x1000px, JPG, menos de 5MB.
