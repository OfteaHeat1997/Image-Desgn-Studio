Tech Stack

Todo lo que se usa para construir UniStudio, desde el frontend hasta las APIs externas.

F
Frontend
  • Next.js 16.1.6 — App Router, Server Actions (50MB max)
  • React 19.2.3 — Ultimo version estable
  • TypeScript — Tipado estricto en todo el proyecto
  • Tailwind CSS 4 — Tema oscuro luxury (#09090B + #C5A47E dorado)
  • Zustand 5 — Estado global (6 stores)
  • Radix UI — Dialog, Select, Slider, Tabs, Tooltip
  • Lucide React — Iconos (100+ usados)
B
Backend & APIs
  • Replicate — 10+ modelos IA (BG remove, video, try-on, etc.)
  • fal.ai — 4 video providers (LTX, Wan 2.5, Kling, Minimax)
  • Anthropic Claude — AI Agent planning + prompt optimization
  • FASHN API — Virtual try-on especializado
  • Edge TTS — Text-to-speech gratuito (Node.js)
  • Sharp 0.34.5 — Procesamiento de imagen server-side
D
Base de Datos & Storage
  • Prisma 7.4.1 — ORM con adaptador PostgreSQL
  • PostgreSQL — Base de datos principal (Docker local)
  • localStorage — Persistencia de gallery-store y video-store
  • File System — Auto-save a /output/{module}/
C
Client-Side Processing
  • @imgly/background-removal — BG removal 100% en browser (WASM, gratis)
  • Fabric.js 7 — Canvas editor con capas
  • OffscreenCanvas — Post-processing local (sombras, resize, formatos)
  • Ken Burns — Animacion de video CSS (gratis, sin API)

Los 18 Modulos Todos Funcionales

Cada modulo con su descripcion, por que lo necesitas, como funciona, y cuanto cuesta. Organizados en 6 categorias.

FONDOS (Backgrounds)

✂️
Quitar Fondo
Gratis-$0.01
Elimina automaticamente el fondo de cualquier foto de producto. 3 proveedores: Navegador (gratis, WASM), Docker local (gratis), Replicate ($0.01).
Por que lo necesitas Amazon, Shopify y la mayoria de marketplaces requieren fondo blanco. Ahorra horas de edicion manual en Photoshop.
Upload → Elegir provider → Tipo salida → Resultado PNG

API: /api/bg-remove → src/lib/processing/bg-remove.ts + bg-remove-browser.ts

🖼️
Fondos con IA
$0.003-$0.05
Reemplaza el fondo con escenas profesionales. 25+ presets (estudio, naturaleza, lujo, belleza, temporada) + prompt personalizado con Claude IA.
Por que lo necesitas Fotos lifestyle venden 30% mas que fondo blanco. Crea escenas de lujo sin estudio fotografico.
Upload → Elegir fondo → Replicate IA → Composicion

API: /api/bg-generate → src/lib/processing/bg-generate.ts

MEJORA (Enhancement)

✨
Mejorar Calidad
Gratis
Ajusta brillo, contraste, nitidez y color. 10 presets profesionales (E-Commerce, Moda, Lujo, Vintage, Nitido). 100% local con Sharp.
Por que lo necesitas Fotos de celular salen oscuras o descoloridas. Este modulo las convierte en calidad profesional.

API: /api/enhance → src/lib/processing/enhance.ts (Sharp, 100% local)

☀️
Sombras e Iluminacion
Gratis-$0.05
5 tipos de sombra: Drop Shadow, Contacto, Reflejo, AI Relight, AI Kontext. Las basicas son gratis (Sharp), las IA usan Replicate.
Por que lo necesitas Sin sombras, los productos parecen flotando. Las sombras dan profundidad y realismo profesional.

API: /api/shadows → src/lib/processing/shadows.ts

↔️
Extender Imagen
$0.05
Expande los bordes para adaptarla a cualquier formato: 12 presets (Amazon 1:1, IG Stories 9:16, Pinterest 2:3, etc.).
Por que lo necesitas Instagram requiere 1:1, Stories 9:16, Amazon 1:1. No recortes — extiende con IA.

API: /api/outpaint → Replicate (Kontext)

EDICION

🧹
Borrar y Reemplazar
$0.03-$0.05
Elimina objetos no deseados o cambia colores con texto. 8 presets rapidos (quitar etiqueta, arrugas, mancha, cambiar color).
Por que lo necesitas Quitar etiquetas, manchas, arrugas o cambiar el color del producto sin re-fotografiar.

API: /api/inpaint → Replicate (Flux Fill / Kontext)

✅
Verificar Marketplace
Gratis
Verifica que tu foto cumple los requisitos de Amazon, Shopify, Etsy, eBay. Detecta problemas de tamanio, formato, resolucion con auto-correccion.
Por que lo necesitas Fotos rechazadas = productos no publicados = ventas perdidas.

API: Local — analisis de metadata de imagen

MODELOS (Models & Try-On)

👕
Prueba Virtual
$0.02-$0.05
Coloca tu prenda sobre un modelo virtual. 3 proveedores IA (IDM-VTON, Kolors, FASHN) con routing inteligente por tipo de prenda.
Por que lo necesitas Fotos en modelo venden 40% mas. Prueba diferentes modelos sin contratar fotografos.

API: /api/tryon → Replicate (IDM-VTON, Kolors) + FASHN

👤
Crear Modelo IA
$0.055
Genera modelos virtuales con genero, edad, tono de piel, pose y expresion personalizables. Para usar con Prueba Virtual.
Por que lo necesitas Diversifica tu catalogo con modelos de todas las etnias sin sesion fotografica.

API: /api/model-create → Replicate (Flux Kontext Pro)

👻
Maniqui Invisible
$0.05-$0.08
Efecto hollow-man profesional. 3 operaciones: quitar maniqui, flat-to-model, model-to-flat.
Por que lo necesitas Los maniquis distraen al comprador. El efecto invisible es estandar en moda profesional.

API: /api/ghost-mannequin → Replicate

💎
Joyeria Virtual
$0.05
Prueba aretes, collares, anillos o pulseras sobre fotos de modelos sin sesion fotografica.
Por que lo necesitas Muestra como se ve la joyeria puesta sin sesion fotografica.

API: /api/jewelry-tryon → Replicate / FASHN

🎬
Video Studio
$0-$0.35

6 Zustand Stores

Estado global de la aplicacion, persistido donde se necesita.

E
editor-store
Canvas layers, selectedLayerId, tool, zoom, canvasWidth/Height, history (50 max), historyIndex. Undo/redo, layer CRUD, filter editing.
V
video-store
Mode, activeTab, selectedProvider, preset, prompt, duration, aspectRatio, avatar settings, TTS config, projects array. Persistido en localStorage.
G
gallery-store
Array de imagenes procesadas con id, filename, URLs, date, operations, project. Persistido en localStorage + DB sync.
B
batch-store
Cola de batch jobs, pipeline definitions, progreso por job, errores.
K
brand-store
Colores de marca, fuentes, logo URL, configuracion de watermark. Sincroniza con /api/brand-kit.
S
settings-store
Preferencias del usuario, idioma, configuraciones globales.

Estado Actual del Proyecto

Resumen de que funciona, que necesita setup, y que falta.

LO QUE FUNCIONA (sin setup adicional):
  • Dashboard con hero animado y 18 modulos
  • Editor con 19 paneles de modulos, todos con logica real
  • Toast notifications (ya no hay alert() feos)
  • Sidebar categorizado con iconos y costos
  • Empty states cuando no hay imagen
  • Descripciones completas en espanol en cada modulo
  • BG Removal en browser (100% gratis, sin API key)
  • Enhance con Sharp (100% gratis, local)
  • Sombras programaticas (drop, contact, reflection — gratis)
  • Fondos de estudio (colores solidos — gratis)
  • Ken Burns video (CSS animation — gratis)
  • Marketplace compliance (analisis local — gratis)
  • Galeria con filtros y preview
  • Brand Kit con colores, fonts, watermark
  • Auto-save de resultados al disco
LO QUE NECESITA API KEYS (en .env.local):
  • Replicate: BG remove cloud, fondos IA, inpaint, outpaint, try-on, model-create, video (Wan), avatar (Wav2Lip, SadTalker)
  • fal.ai: Video (LTX, Wan 2.5, Kling, Minimax), Avatar (MuseTalk, LivePortrait)
  • Anthropic: AI Agent planning, Director Creativo IA (tiene fallback local)
  • FASHN: Try-on especializado (alternativa a Replicate)
LO QUE NECESITA DOCKER/DB:
  • PostgreSQL: Para persistir processing_jobs, brand_kit, gallery history
  • withoutBG Docker: Para BG removal local gratis (opcional — browser mode ya es gratis)
  • Si no tienes DB corriendo, el app funciona pero no persiste historial en la base de datos
BUILD STATUS: npx next build compila exitosamente con CERO errores. El proyecto esta listo para production deployment.
